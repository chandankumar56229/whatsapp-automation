WhatsCamp — WhatsApp Automation CRM with IVR Calling

WhatsCamp is a powerful, self-hosted WhatsApp Business API platform built for modern businesses and agencies. It provides a complete CRM solution that integrates directly with Meta's official WhatsApp Cloud API — zero external dependencies, no per-message platform fees.

Built with Go (backend) and Vue.js (frontend), WhatsCamp is lightweight, blazing fast, and designed for scale. The multi-tenant architecture allows you to manage multiple organizations from a single installation — perfect for digital agencies, SaaS businesses, and enterprises.

Technology Stack

Backend — Go (Golang)

High-performance, compiled language. Zero-dependency standalone binary. REST API with WebSocket support.

Frontend — Vue.js 3

Modern, reactive UI with component-based architecture. Responsive design, real-time updates.

Database — PostgreSQL / SQLite

Flexible database support. SQLite for quick start, PostgreSQL for production scale.

Deployment — Docker

Single Docker command deployment. Docker Compose included for easy setup. Any cloud or VPS.

System Requirements

  • Server: Any Linux VPS (2GB RAM minimum, 4GB recommended)
  • Docker: Docker & Docker Compose
  • Domain: Domain with SSL certificate (for webhook verification)
  • WhatsApp: Meta Business Account + WhatsApp Business API access
